WRIO Internet OS

Internet OS is an operating system designed to run all of its applications and services through a web browser, which is reflected in the project motto: All You Need Is A Browser. All resource hungry operations are processed in the cloud. This allows the system to used run on the simplest thin client devices complying with the minimum hardware requirements with Internet access. WRIO Internet OS is a platform designed to be an upgrade to a distributed, blockchain powered, machine-readable and secure user-centric Web 3.0. The platform serves as a ‘Google Play’ for user-centric apps and predictive services based on our semantic protocol that features automatic data processing and enables accurate auto delivery of content, products and services to relevant consumers without ads.
Features:
  • Distributed web
  • Block-proof and censor-proof
  • Security and privacy
  • Automatic data processing
  • User-centric Web 3.0​ browsing experience

Universal secure sign-in and profile

The blockchain address functions as an ID and stores user’s assets and profile link. The authorization system based on blockchain enables a decentralized global privacy-respectful and secure one-click login for Web 3.0 applications and services. In general, the login process is similar to the one used in social media. This eliminates the need to continuously create new accounts and entrust personal data to services. Personal data may be encrypted, the access is granted through the use of a crypto key stored on the user’s side.
Features:
  • Secure universal sign-in
  • Privacy and anonymity
  • User-controlled disclosure

Web 3.0 pages: a machine-readable semantic data layer

Web 3.0 pages are generated based on vocabularies founded by Google, Microsoft, Yahoo and Yandex, and their indexation does not require search engine optimization or parsing. Data is stored in a machine-readable key:value format. Keys function as a machine dictionary, making folksonomy (tagging) obsolete. One of their distinctive features is automatic data processing, which enables sorting for each user. See details in the tech demo
Pages registered in the blockchain can be searched for using metadata, and the data that has not been encrypted can be used and downloaded from the original source without copy/paste or API — websites’ borders dissolve, and the web turns into an integrated semantic network of reusable data. This enables the development of next-generation predictive search-engines and user-centric services that use blockchain as a unified metadata base.
Features:
  • Secure and structured Linked Data format
  • Machine-readable
  • API free: open, shareable and reusable data

Browser-driven apps

Unlike Web 2.0, Web 3.0 pages UI and functions are determined by the user. This is called a user-centric approach. WRIO cloud apps are a mashup, i.e. apps functions may complement one another in a puzzle-like way.
Cloud apps can operate on the simplest devices, are always up-to-date, do not need to download, install or setup any software. All resource consuming operations are performed on the service provider’s servers, displaying the end result on the user’s screen.
Features:
  • User-centric and predictive UI
  • One-click setup without download
  • Mashup
 Monetization:
  • Cloud/distributed apps marketplace
  • Premium services
  • Percent of in-app purchases

A predictive service example and the platform token

Streams is the first predictive WRIO application that uses pages registered in the blockchain. It sorts pages based on the streams and personalizes them for each user based on their interests and criteria.
Articles are recommended to the most relevant audience and disseminated while there is a positive response from the readers. Parameters that are taken into account include the time of reading, the number donations and donated amounts, commentators activity, number of sharings, etc.

The System rewards users with tokens for various activity that contributes to the project development: the mining of tokens is based on the PoSA concept (proof of social action). The most elegant example is a possibility to receive 10 tokens simply for clicking the “Claim tokens” button once per hour, which encourages user retention. The dissemination and promotion of articles allow users to gain more tokens, while authors get new readers.
Besides, the most active users receive the best Offers (see below) and their own articles are promoted by the system on special terms.
Features:
  • Automatic content delivering
  • Token-stimulated activity
 Monetization:
  • Premium services
  • Tokens sale
  • Percent of subscriptions and premium content purchases
  • Percent of donations
  • Boost and promotion

Rating

In simple words, your Rating is a measure of how active you are on the platform, and how much everyone appreciates your contribution. Unlike an unreliable and easily manipulated 5-star approach, Rating is similar to the PageRank and is calculated automatically based on the users degree of satisfaction that takes into account multiple indicators: the number of sharings, the number of donations and donated amounts, donators’ and followers’ rating, reviews, retention statistics, etc.
The Rating system enables trust with no need in identification. The concept consistency has been proved by other services: the rating buildup is a complicated and time consuming process of earning a reputation that can be easily lost in case of abuses.
Besides, Rating is a social proof of mastering certain skills and knowledge, which, alongside with automatic data processing and delivery, may be applied in multiple services: job hunting and personnel recruitment, the search for like-minded friends, personal and business partners, as an effective anti-spam filter, in promotion where the Rating itself is the best ads, etc. See the White Paper for details.
Features:
  • Trust without identification
  • More informative than any CV
  • Rating is the best advertisement

Brand Loyalty and Offers

Brand Loyalty feature is available to premium services and begins for users with adding the service to bookmarks. The level of loyalty determines the degree and number of benefits for a follower: access to premium content, discounts, prize drawings, and other Offers.
Unlike ads, Offers are voluntarily viewed proposals that can be personalized based on the system’s knowledge of the users, their interests ad preferences. Activity (participation in affiliates’ programs and polls, viewing, sharing, commenting, recommending, etc.) earns discounts and other bonuses. The more frequently a user checks Offers and interacts with them, the better Offers they will get.
Offers are an inbound marketing. We don’t sell inbound — we experience it. An Offer includes an estimate of efficiency expressed as the ratio of user actions (click, share, purchase, comment, etc.) to views. The higher the indicator, the more people will receive the Offer — we call it reactive approach. Thus, the only thing important is the value of the offer to the community, rather than the adverting budget.
Features:
  • Targeted
  • Token-stimulated activity
  • Feedback ads, see White Paper for the details
  • Inbound marketing
  • Reactive approach
 Monetization:
  • Premium
  • Affiliate and other programs
  • Boost and promotion

Vision: a new multi-billion thin client market

The project aims at creating a new web based on automatic data processing, which makes possible multiple unique features described in the Presentation
Mobiles and tablets will be followed by thin client terminal devices with a browser-driven OS. The system can run on any smart devices with Internet access: PCs, mobiles, smart TVs and even modems. They will serve only as a screen with Internet access, and for millions of users they will reveal a custom, curated, secure and viruses-free browsing experience, and eventually they will change the entire web. We will see Web 3.0 version of googles, skypes, ebays, paypals, facebooks, ubers, etc., using a public distributed source of machine-readable data pool. Borders of websites and domains that divide nowadays web will dissolve, and we will witness the advent of the P2P web: a network of interrelated peers powered by WRIO Internet OS that can exchange information without any restrictions, while respecting privacy at the same time.
